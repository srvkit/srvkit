# srvkit

A toolkit for building JavaScript server applications.

- 🌍 Runtime agnostic
- 🧩 Framework agnostic
- 🔌 Vite & Rsbuild plugins
- 🔗 HTTPS & middleware
- 🔥 Live dev server
- 📦 Flexible build targets
